## 建站目的

我希望这个博客不是“展示用的空壳”，而是每天都能使用的学习系统。

每篇文章尽量回答四个问题：

- 今天解决了什么问题？
- 我用了哪些命令、工具或资料？
- 中间踩了哪些坑？
- 下次遇到类似问题如何更快解决？

## 内容方向

- Java 全栈：Java、Spring、Maven、Gradle、MySQL。
- 工程工具：Git、GitHub、Vercel、Neovim、终端配置。
- AI 与系统：PyTorch、遥感 VQA、操作系统、编译原理。
- 个人成长：学习计划、复盘、项目总结。

## 写作格式

```text
目标 -> 过程 -> 命令 -> 结果 -> 复盘
```

先追求稳定记录，再追求表达质量。
